# Dotfiles
- Use with stow
## Requirements
### nvim:
- nvim 0.12^
- Manually run make for [telescope-fzf-native](https://github.com/nvim-telescope/telescope-fzf-native.nvim) and [L3MON4D3/LuaSnip](https://github.com/L3MON4D3/LuaSnip) inside by running:
    ```bash
    chmod +x nvim-make.sh
    ./nvim-make.sh
    ```
